# React Performance: Beyond the Basics

You've memoized your components and virtualized your lists. What's next? Let's dive deeper.

## Understanding the Fiber Architecture

React's Fiber reconciler changed everything. Understanding how it works helps you write faster components.

## State Colocation

Keep state as close to where it's used as possible. Distant state means unnecessary re-renders across your component tree.

## The Compiler is Coming

React Forget (the auto-memoizing compiler) will change how we think about re-renders. But until then, understanding these patterns remains crucial.

Performance optimization is a journey, not a destination.
