# The Psychology of Code Reviews

Code reviews are not just about catching bugs. They're a social contract and a teaching tool.

## Egoless Programming

Leave your ego at the door. The code is not you, and feedback is not personal. This mindset transforms reviews from conflict into collaboration.

## The Sandwich Method

Lead with something positive, address the issue, end with encouragement. It's simple but effective.

## Reviewing for Understanding

Don't just look for bugs — ask "does this make sense?" Code that's correct but incomprehensible is technical debt.

Great code reviews build great teams.
